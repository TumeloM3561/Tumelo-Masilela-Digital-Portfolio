# Network Design Project

## 📖 Description
This project involved designing and configuring a small business network using Cisco Packet Tracer.

## ⚙️ Technologies Used
- Cisco Packet Tracer
- Networking protocols

## 🛠️ Installation
1. Open Cisco Packet Tracer
2. Load the .pkt file

## ▶️ Usage
- Run simulation mode
- Test connectivity using ping
